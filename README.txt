INTALAÇÃO

    Primeiramente é necessário a instalação de um servidor web apache.
Em seguida, basta copiar os arquivos do projeto na pasta do servidor Web e iniciá-lo.

Exemplo para servidor Web Apache no Linux:
    Em geral os arquivos desse tipo de servidor localizam-se na pasta /var/www

www no zip para a pasta /var/www do servidor.
